## Using

### Creating and running bot

```python
import disspy

bot = disspy.DisBot(token="YOUR_TOKEN", prefix="!")

bot.run()
```

### bot.on("ready")

```python
import disspy

bot = disspy.DisBot(token="YOUR_TOKEN", prefix="!")


@bot.on("ready")
async def on_ready():
    print("Ready!")


bot.run()
```
